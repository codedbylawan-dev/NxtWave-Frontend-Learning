## **Modulus Operator (%)**

Used to find the **remainder** of a division.

### Syntax:

```
a % b
```

### Examples:

```python
print(6 % 3)
```

**Output:**

```
0
```

```python
print(7 % 4)
```

**Output:**

```
3
```

---

## **Exponent Operator (**)\*\*

Used to calculate **a to the power b**.

### Syntax:

```
a ** b
```

### Example:

```python
print(2 ** 4)
```

**Output:**

```
16
```

---

## **Square of a Number**

You can find the square of a number by keeping the exponent as **2**.

### Example:

```python
print(5 ** 2)
```

**Output:**

```
25
```

---

## **Square Root of a Number**

You can find the **square root** of a number by using exponent **0.5**.
Any number raised to **0.5** gives its square root.

### Example:

```python
print(16 ** 0.5)
```

**Output:**

```
4.0
```
