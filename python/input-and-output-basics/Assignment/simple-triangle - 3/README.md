# **Simple Triangle – Solved**

### **Question:**

Write a program that prints a simple triangle using stars (\*).

---

## **Input**

(No input for this problem.)

---

## **Output**

```
*
* *
* * *
```

---

## **Explanation**

We need to print a **triangle of stars** with:

- **1 star** on the first line
- **2 stars** on the second line
- **3 stars** on the third line

All stars must be separated by spaces.

---

# **Approach**

### **Step 1:** No input needed

Pattern is fixed.

### **Step 2:** Prepare each line

- Line 1 → `*`
- Line 2 → `* *`
- Line 3 → `* * *`

### **Step 3:** Print each line

Use three print() statements.

---

# **Step-by-Step Explanation**

### **Step 1: No input**

No input is required.

### **Step 2: Create each pattern line**

```python
line1 = "*"
line2 = "* *"
line3 = "* * *"
```

### **Step 3: Print the lines**

```python
print(line1)
print(line2)
print(line3)
```

---

# **Solution (Final Code)**

```python
print("*")
print("* *")
print("* * *")
```
